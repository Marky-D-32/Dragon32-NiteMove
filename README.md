# NiteMove
A machine code game for the dragon 32.  Move the Knight around a chess board.

This game was orginally developed by Pat McCabe and Colin Turner.  It was published in the May 1986 issue of Dragon User.

| File | Description |
|------------------------------|--------------------------------------------------------------------------------------------------------------|
| NiteMove_AllDream.cas        | The Alldream assembly code file |
| NiteMove_Assembly_Code.txt   | An export of the 6809 assembly code used to generate the machine code    |
| NiteMove.cas                 | The game file which can be loaded into the XROAR emulator       |

To run the game:
* Download NiteMove.cas to your local drive
* Paste the following URL into your browser:  https://www.6809.org.uk/xroar/online/?machine=dragon32
* Click on the File tab; Click Load;  Select the downloaded NiteMove.cas file.
* Within the emulator (i.e. on the green screen)
  - Type CLOADM
  - Type EXEC
